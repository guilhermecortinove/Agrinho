Perguntas Frequentes (FAQ)

Clique nas perguntas abaixo para ver as respostas sobre o regulamento:

Quem pode participar do Concurso de Programação?

Estudantes dos Ensinos Fundamental (Anos Finais) e Médio da rede pública estadual do Paraná, desde que estejam matriculados nos componentes de tecnologia/matemática e tenham concluído no mínimo 2 unidades na plataforma Alura.

Qual é o prazo final para enviar o projeto?

Os projetos devem ser enviados pelo ambiente virtual da Alura até as 23h59 do dia 14 de junho de 2026. Após esse horário, nenhuma alteração pode ser feita no código.

Quais são as regras para a Subcategoria 3 (Ensino Médio)?

O projeto deve ser feito em HTML, CSS e JavaScript puros (sem frameworks). Deve ser enviado o link do repositório do GitHub (público) e o site deve estar rodando obrigatoriamente no GitHub Pages ou Vercel.

Posso usar CSS ou JavaScript dentro do arquivo HTML?

Não! O regulamento proíbe terminantemente programação inline ou interna. Todos os estilos devem estar no arquivo style.css e toda a lógica no arquivo script.js.
